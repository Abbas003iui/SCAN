Light Spectrum Analyzer

Instructions:
  1. First, upload a blank/reference image (e.g., empty cuvette or white background) by selecting a file in the "Upload Blank" input.
  2. Next, upload the sample image (containing the spectrum to analyze) using the "Upload Sample" input.
  3. The analyzer calculates absorbance as A = log10(Iblank / Isample), normalizes it, and plots the spectrum.
  4. The graph's colored area under the curve matches the visible light spectrum for corresponding wavelengths (380-750 nm).
  5. The peak wavelength and absorbance value are shown above the graph.

Note: Upload images with similar dimensions for best results.
